# go-attack

A personal security research lab where I simulate offensive security techniques, drop them into isolated VMs (VirtualBox)
, and conduct end-to-end forensics (memory, disk, host artefacts, network, etc,.)

## Disclaimer

All simulations in this repository are:
- Built and tested in an isolated VM environment only
- Intended purely for educational and research purposes
- Never to be used against systems you do not own
